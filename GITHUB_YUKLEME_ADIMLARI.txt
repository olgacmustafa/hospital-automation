GITHUB'A YÜKLEME ADIMLARI

1) GitHub'da yeni repo oluştur.
   Önerilen repo adı: hastane-otomasyonu
   İlk etapta Private seçmen daha güvenli olur.

2) Bu klasörde terminal / Git Bash aç.

3) Şu komutları sırayla çalıştır:

git init
git add .
git status
git commit -m "Initial hospital automation project"
git branch -M main
git remote add origin https://github.com/KULLANICI_ADIN/hastane-otomasyonu.git
git push -u origin main

4) KULLANICI_ADIN kısmını kendi GitHub kullanıcı adınla değiştir.

5) git status çıktısında ortak/baglanti.php görünmemeli.
   Çünkü gerçek bağlantı dosyası GitHub'a gönderilmemeli.
